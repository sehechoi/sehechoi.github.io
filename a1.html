Sehe Choi's Assignment 1: Blink!

Schematic

The circuit schematic shows three 220Ω resistors (R1, R2, and R3) connected to three red LED lights (LED 1, LED 2, and LED 3) and digital pins 8, 12, and 13 on an Arduino Uno. To the right of the schematic are the calculations for resistance, given 5V of power and a desired current of 20mA or less. We assume that there is a voltage drop of 1.8V across a single red LED. The calculations showed that a resistance of 160Ω or greater for each LED would be appropriate for the circuit.

Circuit

The circuit utilizes three digital pins to ensure that each resistor/LED pair gets 5V of power and is able to operate independently. All components are connected to ground using the negative rail of the breadboard.

Firmware

        
          // the setup function
          void setup() {
            // initialize digital pin 8 as an output.
            pinMode(8, OUTPUT);
            // initialize digital pin 12 as an output.
            pinMode(12, OUTPUT);
            // initialize digital pin 13 as an output.
            pinMode(13, OUTPUT);
          }

          // the loop function
          void loop() {
            digitalWrite(8, HIGH);  // turn the LED on by making the voltage of digital pin 8 HIGH (5V)
            delay(1000);                      // wait for a second
            digitalWrite(8, LOW);   // turn the LED off by making the voltage LOW (0V)
            delay(1000);                      // wait for a second

            digitalWrite(12, HIGH);  // turn the LED on by making the voltage of digital pin 12 HIGH
            delay(1000);                      // wait for a second
            digitalWrite(12, LOW);   // turn the LED off by making the voltage LOW
            delay(1000);                      // wait for a second

            digitalWrite(13, HIGH);  // turn the LED on by making the voltage of digital pin 13 HIGH
            delay(1000);                      // wait for a second
            digitalWrite(13, LOW);   // turn the LED off by making the voltage LOW
            delay(1000);                      // wait for a second
          }

Operation

The LEDs are programmed to blink one at a time, but the code could be altered to make them blink in a different order or speed. For example, the code in the firmware section delays blinking by 1000ms, but in the GIF, there is only a 100ms delay.
